Get up and running with Vim in under 5 minutes. This guide covers the absolute essentials you need to start editing files.

## Start Vim

Open a file with Vim:

<CodeGroup>
```bash Terminal
vim filename.txt
```

```bash Create new file
vim newfile.txt
```

```bash Open at specific line
vim +42 file.txt
```
</CodeGroup>

## Understanding modes

Vim has different modes for different tasks:

| Mode | Purpose | How to enter |
|------|---------|--------------|
| **Normal** | Navigate and execute commands | Press `Esc` |
| **Insert** | Type and edit text | Press `i` |
| **Visual** | Select text | Press `v` |
| **Command-line** | Run Ex commands | Press `:` |

<Tip>
When in doubt, press `Esc` to return to Normal mode.
</Tip>

## Basic editing workflow

<Steps>
  <Step title="Open a file">
    ```bash
    vim practice.txt
    ```
  </Step>

  <Step title="Enter insert mode">
    Press `i` to start typing. You'll see `-- INSERT --` at the bottom.
  </Step>

  <Step title="Type some text">
    Type anything you want. Use arrow keys or normal keyboard to edit.
  </Step>

  <Step title="Return to normal mode">
    Press `Esc` to exit insert mode.
  </Step>

  <Step title="Save your work">
    Type `:w` and press `Enter` to write (save) the file.
  </Step>

  <Step title="Exit Vim">
    Type `:q` and press `Enter` to quit.
  </Step>
</Steps>

## Essential commands

### Opening and saving

```vim
:e filename     " Open (edit) a file
:w              " Write (save) current file
:w filename     " Save as new filename
:wq             " Write and quit
:q              " Quit (fails if unsaved changes)
:q!             " Quit without saving
```

### Navigation

```vim
h, j, k, l      " Left, down, up, right
w               " Jump to start of next word
b               " Jump to start of previous word
0               " Jump to start of line
$               " Jump to end of line
gg              " Jump to start of file
G               " Jump to end of file
```

### Insert mode

```vim
i               " Insert before cursor
a               " Insert after cursor (append)
I               " Insert at start of line
A               " Insert at end of line
o               " Open new line below
O               " Open new line above
```

## Your first edit

Let's practice with a real example:

<Steps>
  <Step title="Create a shopping list">
    ```bash
    vim shopping.txt
    ```
  </Step>

  <Step title="Add items">
    Press `i`, then type:
    ```
    Shopping List
    - Apples

    - Bread
    ```

  </Step>

  <Step title="Save">
    Press `Esc`, then type `:w` and press `Enter`
  </Step>

  <Step title="Add more items">

    ```

    ```
  </Step>

  <Step title="Save and exit">
    Press `Esc`, then type `:wq` and press `Enter`
  </Step>
</Steps>

## Common pitfalls

<AccordionGroup>
  <Accordion title="How do I exit Vim?">
    Press `Esc` to enter Normal mode, then:
    - `:q` to quit (if no changes)
    - `:q!` to quit without saving
    - `:wq` to save and quit
  </Accordion>

  <Accordion title="Why can't I type?">
    You're probably in Normal mode. Press `i` to enter Insert mode and start typing.
  </Accordion>

  <Accordion title="I pressed the wrong key!">
    Press `Esc` to return to Normal mode, then press `u` to undo.
  </Accordion>

  <Accordion title="Commands don't work">
    Make sure you're in Normal mode (press `Esc`) before entering commands. Command-line commands start with `:`.
  </Accordion>
</AccordionGroup>

## Help system

Vim has extensive built-in help:

```vim
:help               " Open help
:help i_CTRL-R      " Help for insert mode CTRL-R
:help :write        " Help for :write command
:help 'number'      " Help for 'number' option
```

Press `CTRL-]` on a tag to jump to it, `CTRL-O` to jump back.
