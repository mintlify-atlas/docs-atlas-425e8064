Ex commands are entered by typing `:` followed by the command name. This reference lists all Ex commands available in Vim.

## Command Basics

- Ex commands start with `:` in Normal mode
- Many commands accept a [range](/user-guide/ranges) before the command name
- Commands can be abbreviated to their shortest unique form (shown in brackets)
- Use `|` to separate multiple commands on one line
- Use `"` at the start of a line or after a command to add comments

## Special Characters

In Ex commands:
- `%` - Current file name
- `#` - Alternate file name
- `<cword>` - Word under cursor
- `<cWORD>` - WORD under cursor
- `<cfile>` - File path under cursor
